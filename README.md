# UseThis - Peer-to-Peer Student Rental Platform

UseThis is a comprehensive peer-to-peer rental platform designed specifically for university students to share resources, build community, and earn money within their campus networks.

## What It Does

**Core Purpose**: Enables students to rent items from each other - from textbooks and electronics to furniture and sports equipment - creating a sustainable campus economy.

**Key Features**:
- **Item Listings**: Students can list items they own for rent with photos, descriptions, and pricing
- **Smart Browse**: Search and filter available items by category, price, location, and availability
- **Booking System**: Request rentals with flexible date ranges and instant notifications
- **Secure Messaging**: Built-in chat system for coordinating pickups and discussing items
- **Review System**: Rate and review both renters and item owners to build trust
- **Profile Management**: Comprehensive user profiles with rental history and ratings

## Technology Stack

- **Frontend**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router
- **Backend**: Supabase (authentication, database, real-time features)
- **Deployment**: Netlify

## Current Status

This is a **fully functional demo** showcasing all core features. The ultimate vision is to create university-specific instances where students can only join using their official .edu email addresses, creating trusted campus communities with verified student networks.

**Live Demo**: http://use-this.netlify.app/

The platform demonstrates how students can transform campus resource sharing from informal arrangements into a structured, safe, and efficient marketplace.
